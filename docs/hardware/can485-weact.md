# WeAct Studio ESP32 CAN485

The WeAct Studio ESP32 CAN485 board is compatible with the MOT ESP32-WROOM firmware path when it exposes the SN65HVD230 on the same GPIOs.

```text
CAN transceiver: SN65HVD230
CAN RX GPIO:     32
CAN TX GPIO:     13
CAN bitrate:     500 kbit/s
Mode:            TWAI receive-only
```

The RS485 part of the board is not used by MOT.
