# LilyGO T-A7670G

LTE-capable MOT platform.

## Features

- SIMCom A7670G LTE modem
- External L76K GPS module
- CAN via SN65HVD230
- Local AP and WebUI
- OTA
- MQTT over LTE

## LTE stack

Uses the LewisXhe TinyGSM A76XXSSL fork.
