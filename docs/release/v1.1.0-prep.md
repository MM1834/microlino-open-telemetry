# v1.1.0 Release Preparation

## Release goal

Promote the current develop branch into a documented main release.

## Required checks

### ESP32-WROOM

- Build succeeds
- Flash succeeds
- Web UI works
- CAN decoder works
- MQTT works
- ABRP works
- Backup / Restore works
- OTA works

### LilyGO T-A7670G R2

- Build succeeds
- Flash succeeds
- WiFi works
- LTE modem registers
- LTE IP available
- GPS fix visible when available
- CAN frames received
- Decoder updates telemetry
- MQTT over WiFi works
- LTE MQTT field test pending or documented
- ABRP over WiFi works
- Backup / Restore works
- OTA works

## Known issues

- LilyGO modem may require full power-cycle if stuck.
- LilyGO LTE MQTT is experimental until field-tested.
- LilyGO ABRP LTE transport is not final.
- ESP32-WROOM does not send lat/lon without GPS.

## Release steps

```bash
git checkout develop
git status
pio run
# test both firmware targets

git checkout main
git pull origin main
git merge --no-ff develop
git tag -a v1.1.0 -m "Microlino Open Telemetry v1.1.0"
git push origin main
git push origin v1.1.0
```
