# v1.1.0 LilyGO Stability Release

## Included

- LewisXhe TinyGSM A76XXSSL transport for A7670G-LLSE.
- LTE modem initialization and GPRS/PDP connection.
- MQTT over LTE confirmed working.
- Backup/Restore confirmed working.
- External L76K GPS status and telemetry.
- CAN receive diagnostics.
- OTA and WebUI retained.
- ABRP over WiFi retained.

## Deferred

- ABRP over LTE HTTPS is intentionally disabled in this stability release.
- Failed LTE TLS attempts blocked the WebUI because modem/TinyGSM operations share the same UART and loop.
- ABRP LTE HTTPS will be handled in a follow-up sprint with isolated secure-client testing and backoff/non-blocking behavior.

## Validation

```bash
cd firmware/lilygo-t-a7670
pio run -t clean
pio run
pio run -t upload
pio run -t uploadfs
```

Runtime checks:

- WebUI loads after uploadfs.
- Factory reset works.
- Restore imports JSON config.
- LTE modem shows IP.
- MQTT over LTE connects and publishes.
- GPS shows valid coordinates after fix.
- CAN diagnostics page loads.
- ABRP over LTE does not auto-run/block WebUI.
