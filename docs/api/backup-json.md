# Backup JSON

Example:

```json
{
  "deviceName": "mot-lilygo-fe8ce0",
  "vehicleId": "pioneer",
  "mqttPrefix": "mot",
  "wifiSsid": "",
  "lteApn": "gprs.swisscom.ch",
  "mqttHost": "example.local",
  "mqttPort": 1883,
  "otaEnabled": true,
  "abrpEnabled": true
}
```

Sensitive fields include passwords, MQTT credentials and ABRP tokens.
