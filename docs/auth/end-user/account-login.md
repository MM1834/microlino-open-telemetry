# User account and login

## Delivery status

SPR-0004B.1.1 creates only the secure user directory. Registration and login are not available in the dashboard yet.

A later increment will provide:

- sign-in with email address and password;
- first-login password change;
- email verification;
- password recovery;
- sign-out.

A user account alone will not grant access to a vehicle. Vehicle permissions are managed separately.
