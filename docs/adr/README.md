## Legacy ADRs

Legacy ADRs using the historic numbering (`ADR-0001`, `ADR-0002`, …)
remain valid historical records.

They should not be renumbered because that would break references.

Instead, future ADRs continue with the new numbering scheme while this index
provides the bridge between generations.
