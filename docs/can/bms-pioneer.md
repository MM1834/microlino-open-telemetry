# Microlino CAN - BMS 1 / Pioneer

This document is a placeholder for Pioneer BMS CAN decoding.

The BMS CAN profile is planned as an optional second CAN input in future MOT versions.

## Planned Role

Display CAN remains the common baseline.

Pioneer BMS CAN may add values such as:

- battery voltage
- battery current
- battery power
- cell voltages
- temperatures
- BMS status

## Status

Not implemented in the current WROOM reference firmware.
