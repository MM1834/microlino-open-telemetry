# Microlino CAN - BMS 2 / Standard CAN

This document is a placeholder for newer Microlino BMS CAN decoding.

The BMS 2 profile is planned as an optional second CAN input in future MOT versions.

## Planned Role

Display CAN remains the common baseline.

Standard BMS CAN may add values such as:

- battery voltage
- battery current
- battery power
- temperature values
- charging state
- cell data

## Status

Not implemented in the current WROOM reference firmware.
